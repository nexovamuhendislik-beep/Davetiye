# Gülsena & Yusuf — Davetiye

[Tilda Template 6](https://webgency.tilda.ws/template6) tarzında dijital düğün davetiyesi.

## Özellikler

- **Click to open** — animasyonlu zarf açılışı
- **Video arka plan** (opsiyonel)
- **Scratch to reveal** — kazıyarak tarih görme (Kına + Düğün)
- Program akışı, mekan, kıyafet kodu
- RSVP formu
- Arka plan müziği (opsiyonel)

## Geliştirme

```bash
npm install
npm run dev
```

http://localhost:5173

## İçerik

Tüm metinler `src/config.ts` dosyasında.

## Medya

- Müzik: `public/music.mp3` → `musicSrc: "/music.mp3"`
- Video: `public/video.mp4` → `videoSrc: "/video.mp4"`
